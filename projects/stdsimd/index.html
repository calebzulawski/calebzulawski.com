Rust Portable SIMD Project Group

View on GitHub

I’ve been a member of the Rust Portable SIMD Project Group since September 2020. Our charter is to create the std::simd module for the Rust programming language’s standard library, providing a safe, easy, and portable abstraction for using SIMD. We are currently in the process of stabilizing the project for inclusion in the nightly compiler.
